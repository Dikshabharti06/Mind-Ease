# MindEase 🧠

MindEase is a full stack mental wellness platform designed to spread awareness about mental health and provide users with a supportive, accessible, and user-friendly digital space. The platform focuses on promoting mental well-being through interactive features and informative resources.

## 🚀 Features
- User-friendly mental wellness interface
- Mental health awareness resources
- Responsive and interactive design
- Organized content sections
- Backend integration for dynamic functionality
- Clean and accessible UI

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB
